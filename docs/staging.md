# Staging

GitPad mirrors Git's index workflow.

## Operations

- Stage file
- Unstage file
- Stage all
- Unstage all
- Discard file changes
- Discard all unstaged changes

## Safety

- Discard actions require confirmation.
- Conflicted files are tracked separately and shown in the source control panel.
- Commit button is disabled when no staged files are present.

## Commit Message

A message is required for every staged commit.
