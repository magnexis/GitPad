# Shortcuts

- `Ctrl+K` / `Cmd+K`: Open command palette
- `Ctrl+Shift+P`: Open command palette
- `Ctrl+P` / `Cmd+P`: Quick open file
- `Ctrl+S` / `Cmd+S`: Save active file
- `Ctrl+W` / `Cmd+W`: Close active tab
- `Ctrl+B` / `Cmd+B`: Toggle explorer sidebar
- `Ctrl+Shift+F` / `Cmd+Shift+F`: Focus search view
- `Ctrl+N` / `Cmd+N`: Create new note
- `Ctrl+Shift+N` / `Cmd+Shift+N`: Quick capture modal
- `Escape`: Close palette/quick-open/modal prompts
- `Arrow Up/Down`: Navigate command and quick-open results
- `Enter`: Execute selected command/result
