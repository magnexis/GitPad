# Features

## Workspaces

- Create, open, switch, rename, delete.
- Last-opened workspace restored from persisted ordering/state.
- Workspace creation initializes Git repository without committing.

## Explorer

- Hierarchical folders/files.
- Create file/folder, rename, duplicate, delete.
- Drag-and-drop move.
- Right-click context menu.
- Collapse all.
- Explorer filter/search.

## Editor

- Multi-tab file editing.
- Dirty state tracking.
- Save active and save all.
- Close tab, close others, close all, reopen closed tab.
- Markdown preview and code execution output.

## Git and History

- Branch selector and branch management APIs.
- Source control panel with staged/unstaged/conflicted sections.
- Stage/unstage/discard workflows.
- Commit history list and filters.
- Commit detail inspection with changed file list.
- Revert commit and restore file from commit.
- Diff views from history and source control panel.

## Productivity

- Command palette (`Ctrl+K`) with categorized actions.
- Quick open (`Ctrl+P`) for files.
- Keyboard-first operations across explorer/editor/workspace actions.

## Accounts and Activity

- Local register/login/logout/session.
- Account panel with known users.
- Activity feed panel for auth/file/git events.

## Search

- Full-text search with filters.
- Search-inside-results refinement.
- Highlighted matches and direct open.
